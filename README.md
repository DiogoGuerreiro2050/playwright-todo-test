# TodoMVC Playwright Test

https://nodejs.org/en/download

## Install nodejs

```bash
https://nodejs.org/en/download
```

## Setup

```bash
npm install
```

## Run the Test

```bash
npx playwright test
```

## View the Report

```bash
npx playwright show-report
```

Outputs:
- Screenshots → `screenshots/`
- Video & trace → `test-results/`
